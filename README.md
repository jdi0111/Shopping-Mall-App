# Shopping-Mall-App
## FILES

- `Ai` : Ai 파일들이 있습니다
- `PSD`: PSD 파일들이 있습니다
- `SCREENSHOTS`: 스크린샷이 저장되어 있습니다

- 메인화면 타이틀  `#ffffff`
- 메인화면 목차 `#ffe0e0`
- 네비게이션드로어 `#f07878` ~ `#ffafaf`
